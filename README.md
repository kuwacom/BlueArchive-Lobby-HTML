# BlueArchive-Lobby-HTML
BlueArchiveのメモロビをHTMLでへ実装しブラウザで閲覧できるようにしたもの

ビルド出力されたHTMLをリリースバージョンごとにコミットしていきます

# 利用ライブラリ
- Core: React + Vite
- UI: MUI
